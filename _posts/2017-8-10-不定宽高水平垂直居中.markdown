##实现代码如下
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Flexbox布局</title>
    <style type="text/css">
        .box{
	    display:-webkit-flex;
	    position:relative;
	    width:200px;
	    height:200px;
	    background:#ccc;
	}
	.child{
	    position: absolute;
	    top:50%;
	    left:50%;
	    transform: translate(-50%,-50%);
	    background:#fff;
	}
    </style>
</head>
<body>
    <div class="box">
	this is parent!
	<div class="child">
	    child
	</div>
    </div>
</body>
</html>
```

备注：CSS3中的translate属性值为百分比时，是相对于元素自身的宽高。
