#  Pixel 移动开发像素知识
###px：CSS pixels 逻辑像素，浏览器使用的抽象单位

###dp,pt: device independent pixels 设备无关像素

###dpr: devicePixelRatio 设备像素缩放比


###计算公式：1px = (dpr)² * dp

为什么iphone5是320px * 568px -> 因为dpr = 2

Retina屏（高清屏）: dpr都是大于等于2
